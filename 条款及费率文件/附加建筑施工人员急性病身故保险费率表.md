# 附加建筑施工人员急性病身故保险费率表

# 一、基准赔付标准和基准保险费

# （一）基准赔付标准

基准每人保险金额：1万元

# （二）基准赔付标准对应的基准保险费

1.按工程造价计收保险费（非农村建房适用）

基准保险费率：工程总造价的  $0.006 \text{‰}$

2.按建筑面积计收保险费（仅农村建房适用）

基准保险费（每平方米）：0.015元

# （三）基准保险费计算

1.按工程造价计收保险费

基准保险费  $=$  基准费率  $\times$  工程总造价  $\times$  （每一被保险人保险金额  $\div 10,000$ ）

2.按建筑面积计收保险费

基准保险费  $=$  建筑施工总面积（平方米）  $\times$  每平方米基准保险费  $\times$  （每一被保险人保险金额  $\div 10,000$ ）

# 二、费率调整系数

费率调整系数为以下各项系数之乘积，当某项调整系数相关风险信息不确定时，该系数取1.0。

（一）被保险人风险状况系数

注：A类水平：指从被保险人性别年龄、健康状况、生活方式、安全意识等方面综合判断风险低。  
B类水平：指从被保险人性别年龄、健康状况、生活方式、安全意识等方面综合判断风险中等。  
C类水平：指从被保险人性别年龄、健康状况、生活方式、安全意识等方面综合判断风险高。

# 本调整系数不适用于高速公路、公路、铁路工程。

# 三、保险费计算公式
