# 附加建筑施工人员意外伤害住院津贴保险费率表

# 一、基准给付标准和基准保险费

# （一）基准给付标准

基准免赔日数：3日

基准每次最高给付日数：60日

基准累计承担住院津贴给付日数：180日

基准保险期间届满住院延长津贴给付日数：30日

# （二）基准给付标准对应的基准保险费

每人每日意外伤害住院津贴金额为10元，该产品保险费有2种方式计收：

1. 按照建筑工程项自施工合同总造价计收保险费（非农村建房适用）

基准给付标准对应的基准费率： $0.004 \text{‰}$

2. 按照建筑工程项目施工合同总面积计收保险费（农村建房适用）

基准给付标准对应的每平方米基准保险费：0.016元

# （四）基准保险费计算

# 1.按照建筑工程项目总造价计收保险费

基准保险费  $=$  （每人每日意外伤害住院津贴金额/10）  $\times$  基准给付标准对应的基准费率  $\times$  项目总造价  $\times$  参数调整系数

# 2.按照建筑工程项目总面积计收保险费

基准保险费  $=$  （每人每日意外伤害住院津贴金额/10）  $\times$  基准给付标准对应的每平方米基准保险费  $\times$  项目总面积（平方米）  $\times$  参数调整系数
