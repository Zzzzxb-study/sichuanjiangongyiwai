# 合同解析 → 智能报价映射方案

## 📊 字段映射关系分析

### 一、字段对比表

| 报价模块字段 | 类型 | 合同解析字段 | 映射关系 | 转换规则 |
|------------|------|-------------|---------|---------|
| **projectNature** | 枚举 | `isRuralBuilding` | ✅ 直接映射 | true→RURAL, false→NON_RURAL |
| **baseAmount** | 数字（元） | `projectCost` 或 `buildingArea` | ✅ 需转换 | projectCost(万元)×10000 → baseAmount(元) |
| **contractType** | 枚举 | `contractType` | ⚠️ 需转换 | 字符串值映射 |
| **engineeringClass** | 枚举(1-4) | `engineeringClass` | ✅ 直接映射 | 数字1-4直接对应 |
| **durationDays** | 数字（天） | `duration` | ✅ 直接映射 | 天数直接对应 |
| **qualification** | 枚举 | - | ❌ 无对应 | 需用户选择或默认值 |
| **riskManagementLevel** | 枚举 | - | ❌ 无对应 | 需用户选择或默认值 |
| **coverageAmount** | 数字（元） | - | ❌ 无对应 | 需用户选择或默认值 |

### 二、合同类型映射规则

```typescript
// 合同解析的contractType → 报价模块的ContractType
const CONTRACT_TYPE_MAP: Record<string, ContractType> = {

```

### 三、计算基数映射规则

```typescript
// 优先级：projectCost > buildingArea
const getBaseAmount = (analysisResult: ContractAnalysisResult): number => {
  // 1. 如果是农村自建房，使用建筑面积（area模式）
  if (analysisResult.isRuralBuilding && analysisResult.buildingArea > 0) {
    return analysisResult.buildingArea; // 平方米
  }

  // 2. 非农村自建房，使用工程造价（value模式）
  if (analysisResult.projectCost > 0) {
    return analysisResult.projectCost * 10000; // 万元 → 元
  }

  // 3. 都没有，返回默认值
  return 85000000; // 8500万元（默认）
};
```

## 🎯 映射方案设计

### 方案A：完全自动映射（快速但有局限）

**核心思路**：一键映射，无法确认字段对应关系

**优点**：
- ✅ 操作简单，一键完成
- ✅ 速度快

**缺点**：
- ❌ 用户无法确认映射是否正确
- ❌ 无法调整字段对应关系
- ❌ 可能映射错误（如contractType无法识别时）

**实现方式**：
```typescript
// 在ContractAnalysis.tsx的历史记录表格中
<Button onClick={() => autoMapToPricing(record)}>
  🚀 一键导入报价
</Button>
```

### 方案B：映射预览+确认（推荐⭐⭐⭐⭐⭐）

**核心思路**：显示映射关系预览，用户确认后再导入

**优点**：
- ✅ 用户可以看到所有映射关系
- ✅ 可以调整映射字段
- ✅ 可以修改字段值
- ✅ 有默认值建议

**缺点**：
- ⚠️ 需要额外点击确认
- ⚠️ 需要开发预览界面

**实现方式**：
```typescript
// 1. 点击"导入报价"按钮

<Modal

      { title: '操作', dataIndex: 'action', render: (_, record) => (
          <Button size="small" onClick={() => editMapping(record)}>
```

#### 步骤3：创建映射预览弹窗组件

```typescript
<Modal
  title={
    <Space>
      <CalculatorOutlined />
      <span>导入智能报价 - 确认字段映射</span>
    </Space>
  }
  width={900}
  visible={mappingModalVisible}
  onOk={confirmMapping}
  onCancel={() => setMappingModalVisible(false)}
  okText="确认导入"
  cancelText="取消"
>
  <Alert
    message="请确认以下字段映射关系"
    description="自动映射的字段可以直接使用，未提取的字段已设置默认值，您可以修改后再导入。"
    type="info"
    showIcon
    style={{ marginBottom: 16 }}
  />

  <Table
    columns={[
      {
        title: '报价模块字段',
        dataIndex: 'pricingField',
        width: 150,
        fixed: 'left',
      },
      {
        title: '合同解析字段',
        dataIndex: 'contractField',
        width: 120,
        render: (text, record) => (
          <div>
            <div style={{ fontWeight: 'bold' }}>{text}</div>
            <div style={{ fontSize: '12px', color: '#888' }}>
              {record.contractFieldValue}
            </div>
          </div>
        ),
      },
      {
        title: '映射值',
        dataIndex: 'mappedValue',
        width: 200,
        render: (text, record) => {
          if (record.editable && record.options) {
            return (
              <Select
                style={{ width: '100%' }}
                value={record.mappedValueKey}
                onChange={(value) => updateMappingValue(record, value)}
              >
                {record.options.map(opt => (
                  <Option key={opt.value} value={opt.value}>
                    {opt.label}
                  </Option>
                ))}
              </Select>
            );
          }
          return (
            <Input
              value={text}
              onChange={(e) => updateMappingValue(record, e.target.value)}
              disabled={!record.editable}
            />
          );
        },
      },
      {
        title: '状态',
        dataIndex: 'editable',
        width: 80,
        render: (editable) => (
          <Tag color={editable ? 'blue' : 'green'}>
            {editable ? '可修改' : '已锁定'}
          </Tag>
        ),
      },
    ]}
    dataSource={mappingPreviewData}
    pagination={false}
    scroll={{ y: 400 }}
  />

  <div style={{ marginTop: 16, textAlign: 'right' }}>
    <Space>
      <Text type="secondary">
        💡 提示：已锁定的字段为自动映射，可修改的字段为默认值或未提取字段
      </Text>
    </Space>
  </div>
</Modal>
```
