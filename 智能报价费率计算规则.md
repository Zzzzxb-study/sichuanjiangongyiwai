# 📊 四川建工意外险智能报价费率计算规则

## 🎯 总体说明

系统支持两种计费模式：
- **造价型**（非农村建筑）：按工程造价计算
- **面积型**（农村自建房）：按建筑面积计算

---

## 1️⃣ 主险：意外伤害保险

### 📌 基准费率

| 项目类型 | 基准费率 | 说明 |
|---------|---------|------|
| **造价型** | 0.0436‰ | 每一万元保额 |
| **面积型** | 0.069元/㎡ | 每一万元保额 |

### 📐 费率计算公式

#### 造价型公式：
```
保费 = 工程造价(元) × (每人保额/10000) × 0.0436‰ × K1×K3×K4×K5×K6×K7×(K8)
```

#### 面积型公式：
```
保费 = 工程面积(㎡) × 0.069元/㎡ × (每人保额/10000) × K2×K3×K4×K5×K6×K7×(K8)
```

| 3,000 | 1.00 |
| 6,000 | 0.90 |
| 10,000 | 0.85 |
| 20,000 | 0.75 |
| 50,000 | 0.70 |

> 规律：工程造价越大，系数越小（规模效应）

## 3️⃣ 附加险：住院津贴保险

### 📌 基准费率

| 项目类型 | 基准费率 | 说明 |
|---------|---------|------|
| **造价型** | 0.004‰ | 每日津贴10元为基准 |
| **面积型** | 0.016元/㎡ | 每日津贴10元为基准 |

### 📐 费率计算公式

#### 造价型：
```
保费 = K1×工程造价×(日津贴/10)×0.000004×AM1×AM2×AM3×K3×K4×K5×K6×K7
```

#### 面积型：
```
保费 = K2×工程面积×(日津贴/10)×0.016×AM1×AM2×AM3×K3×K4×K5×K6×K7
```

> 注：(日津贴/10)为津贴金额倍数

### 🔢 调整系数详解

## 5️⃣ 附加险：高原病保险

### 📌 基准费率

**基础加费比例：8.1%**

### 📐 费率计算公式

```
保费 = (主险保费 + 关联险种保费) × 加费比例×R1×R2
```

> 注：以主险和其他关联险种保费之和为基数

### 🔢 调整系数详解

#### **R系列：高原病参数调整系数**

**R1：被保险人风险状况（高原适应力）系数**

| A类 | 0.80 | 高原适应力强 |
| B类 | 1.00 | 高原适应力中等 |
| C类 | 1.20 | 高原适应力弱 |

**R2：区域性（海拔风险）系数**

| 区域等级 | R2系数 | 说明 |
|---------|--------|------|
| A类 | 0.90 | 低海拔风险 |
| B类 | 1.00 | 中海拔风险 |

## 📊 总结对比表

| 险种 | 基准费率(造价型) | 主要调整系数 | 特殊系数 |
|-----|----------------|------------|---------|
| **主险** | 0.0436‰ | K1-K8 | - |
| **医疗保险** | 0.33‰ | M1-M5, MF1-MF7 | - |
| **住院津贴** | 0.004‰ | AM1-AM3, K3-K7 | 津贴倍数 |
| **急性病身故** | 0.006‰ | AQ1-AQ3 | - |
| **高原病** | 8.1%加费 | R1-R2 | 按总保费加收 |

---

## 💡 计算要点总结

1. **所有保费向上取整到元**
2. **线性插值用于连续系数查找**
3. **系数相乘得到总调整系数**
4. **附加险继承主险的大部分全局系数**
5. **高原病保险是加费性质，按总保费百分比计算**
