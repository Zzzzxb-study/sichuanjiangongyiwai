```typescript
projectNature: isRuralBuilding ? 'RURAL' : 'NON_RURAL'
```

- 农村自建房 → RURAL
- 非农村建筑 → NON_RURAL

```typescript
baseAmount: isRuralBuilding

```

```typescript
const CONTRACT_TYPE_MAP = {
```

### 完整流程图

